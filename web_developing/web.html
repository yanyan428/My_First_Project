Web Programming CIS385

SYLLABUS Course Description

This course will introduce students to client-side web programming. Emphasis is placed on HTML, JavaScript and CSS in order to solve elementary level application problems. Students will be assigned web projects that facilitate understating of design and programming concepts. The final project is to create a complete online web-based sales application system.

For the successful completion of this course a student should be able to:

Know the basic structure of HTML. Understand how to format Web pages. Learn how to work with images and tables. Understand design techniques and use Cascading Style Sheets to produce a consistent style in all pages of a Web site. Know the basics of JavaScript. Embed JavaScript programs into HTML code. Learn how to create forms with HTML and JavaScript. Work, read and consume data from APIs.
